## üöÄ Technical Concepts Demonstrated
This repository serves as a practical implementation of the following concepts (see `OOP_Concepts/` folder for isolated examples):

- **Encapsulation:** Protected attributes (`_balance`, `_pin`) and `@property` decorators.
- **Inheritance:** Base `Person` class extended by `Doctor` and `Patient`.
- **Polymorphism:** Overridden methods for different pass types (Student/Staff).
- **Abstraction:** Interface definitions using `ABC` (Abstract Base Classes).
- **Error Handling:** Custom exception hierarchies for robust error management.
